# dhive Widgets

The **dhive** section of HiveFlutterKit provides a rich set of Flutter widgets for building Hive blockchain-powered social and content apps. These widgets cover all major social features: posts, blogs, comments, communities, and user feeds. Each widget is designed for easy integration, infinite scrolling, and customizable callbacks for user interaction.

Below is an overview of the main widgets and their features. For detailed usage and screenshots, see the dedicated documentation pages for each widget.

---

## 📃 Communities List

Display a searchable, infinitely scrollable list of Hive communities. Users can browse, search, and select communities with avatars, descriptions, and member counts.

![Communities List Example](/img/dhive/image-15.png)

---

## 🏠 Community Feed

Show a feed of posts (discussions) from a specific community or tag. Supports sorting (trending, hot, created), infinite scroll, and interaction callbacks for tap, vote, comment, and reblog.

![Community Feed Example](/img/dhive/image-9.png)

---

## 📝 Blog Screen

Display a user's blog (original posts and reblogs) in a familiar blog format. Includes infinite scrolling and customizable interaction callbacks.

![Blog Screen Example](/img/dhive/image-3.png)

---

## 🗨️ Comments Screen

Show a scrollable list of comments authored by a specific Hive account. Infinite scroll and interaction callbacks are supported.

![Comments Screen Example](/img/dhive/image-6.png)

---

## 🧑‍💻 Account Posts Screen

Display all posts by a Hive user in a scrollable list, with support for infinite scroll and all standard post interactions.

![Account Posts Screen Example](/img/dhive/image.png)

---

## 💬 Hive Post Comments

A full-featured comment thread and reply widget for any Hive post or video. Includes search, reply, upvote, and posting new comments.

---

## 🧩 Integration

All widgets are designed for seamless integration in Flutter apps. They use the `HiveFlutterKitPlatform` for blockchain access and provide callbacks for all major user actions.
